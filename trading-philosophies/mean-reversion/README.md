# 均值回归交易哲学

## 核心思想

**"价格终将回归均值"**

均值回归是基于统计学原理的交易哲学。其核心理念是：当价格偏离其长期平均值过远时，会有回归均值的倾向。极端的价格波动往往是暂时的，市场会自我修正。

### 哲学基础

1. **统计回归**：极端值往往会向平均值回归
2. **过度反应**：市场经常对消息过度反应
3. **钟摆效应**：价格像钟摆一样在均值两侧摆动
4. **均值引力**：价格偏离均值越远，回归力量越强

## 战法原理

### 1. 均值定义
- **移动平均线**：20日、50日、200日均线
- **布林带中轨**：价格的统计均值
- **历史价格区间**：一段时间内的平均价格

### 2. 偏离识别
- **标准差**：价格偏离均值2个标准差以上
- **布林带**：价格触及上轨（超买）或下轨（超卖）
- **RSI指标**：RSI>70超买，RSI<30超卖

### 3. 入场时机
- **超买做空**：价格大幅偏离均值上方
- **超卖做多**：价格大幅偏离均值下方
- **确认信号**：等待反转信号确认

### 4. 出场策略
- **回归均值**：价格回到均值附近获利了结
- **反向偏离**：价格向另一方向偏离
- **止损**：价格继续偏离，突破极值

## 历史案例

### 詹姆斯·西蒙斯（Renaissance Technologies）
- **哲学**：统计套利和均值回归
- **案例**：通过量化模型捕捉价格偏离
- **教训**：数学和统计在交易中的应用

### 长期资本管理公司（LTCM）
- **哲学**：债券价差的均值回归
- **案例**：1998年俄罗斯债务危机导致破产
- **教训**：均值回归失效时的风险

## 实战应用

### 适用市场
- ✅ 震荡市场（横盘整理）
- ✅ 成熟市场（波动相对稳定）
- ✅ 配对交易（相关资产）
- ❌ 强趋势市场（会持续亏损）

### 优势
1. **高胜率**：大多数时候价格会回归
2. **明确目标**：均值是清晰的目标位
3. **适合震荡市**：在横盘市场表现优异
4. **可量化**：容易建立数学模型

### 劣势
1. **趋势中失效**：强趋势会导致连续亏损
2. **小赚大赔**：偶尔的趋势会吃掉大部分利润
3. **需要止损**：必须严格止损防止趋势
4. **时机难把握**：何时偏离足够大？

## 心理要求

1. **逆向思维**：敢于在极端时刻反向操作
2. **冷静客观**：不被市场情绪影响
3. **严格纪律**：必须严格执行止损
4. **统计思维**：理解概率和期望值

## 风险管理

1. **严格止损**：防止趋势导致巨亏
2. **仓位控制**：单笔风险不超过1%
3. **避免加仓**：不要在亏损时加仓
4. **识别趋势**：在强趋势时停止交易
