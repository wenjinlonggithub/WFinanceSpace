# 波段交易哲学

## 核心思想

**"捕捉中期价格波动"**

波段交易是介于日内交易和长期投资之间的交易哲学。通过捕捉市场的中期波动（几天到几周）来获利，既避免了日内交易的高频压力，又不需要长期投资的耐心。

### 哲学基础

1. **波动获利**：市场总是在波动中前进
2. **节奏把握**：理解市场的节奏和周期
3. **技术为主**：主要依靠技术分析
4. **灵活应变**：根据市场变化调整策略

## 战法原理

### 1. 时间框架
- **日线图**：主要分析时间框架
- **4小时图**：辅助判断
- **持仓周期**：3-10天

### 2. 波段识别
- **支撑阻力**：在支撑位买入，阻力位卖出
- **形态突破**：三角形、旗形等突破
- **趋势回调**：趋势中的回调机会

### 3. 入场时机
- **突破入场**：突破关键价位
- **回调入场**：趋势回调到支撑
- **形态完成**：技术形态完成

### 4. 出场策略
- **目标位**：预设的风险回报比
- **阻力位**：到达关键阻力
- **趋势反转**：波段结束信号

## 历史案例

### 马克·米勒维尼
- **哲学**：SEPA方法（特定入场点分析）
- **案例**：通过波段交易赢得美国投资冠军
- **教训**：精确的入场点和严格的风险管理

## 实战应用

### 适用市场
- ✅ 波动适中的市场
- ✅ 有明确波段的市场
- ✅ 股票、期货、外汇
- ❌ 极端波动市场

### 优势
1. **时间灵活**：不需要盯盘
2. **风险可控**：有充足时间分析
3. **收益可观**：捕捉中期波动
4. **压力适中**：不像日内交易那么紧张

### 劣势
1. **隔夜风险**：需要承担隔夜风险
2. **需要耐心**：等待波段形成
3. **假突破**：可能遇到假突破
4. **止损较大**：相对日内交易止损更大

## 心理要求

1. **耐心**：等待波段形成和发展
2. **纪律**：严格执行交易计划
3. **灵活**：根据市场调整策略
4. **定力**：不被短期波动影响

## 风险管理

1. **止损设置**：每笔风险1-2%
2. **仓位控制**：根据波段大小调整
3. **分散投资**：同时持有多个波段
4. **风险回报比**：至少1:2
