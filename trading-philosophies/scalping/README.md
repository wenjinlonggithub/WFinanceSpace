# 剥头皮交易哲学

## 核心思想

**"快进快出，积少成多"**

剥头皮（Scalping）是一种超短线交易哲学，通过频繁交易捕捉微小的价格波动来获利。每笔交易的利润很小，但通过高频率和高胜率来累积收益。

### 哲学基础

1. **微利累积**：小利润通过高频率累积成大收益
2. **快速决策**：在极短时间内完成交易决策
3. **高胜率**：追求70%以上的胜率
4. **流动性为王**：只交易流动性最好的品种

## 战法原理

### 1. 时间框架
- **1分钟图**：最常用的时间框架
- **5分钟图**：稍长的剥头皮
- **Tick图**：极短线交易

### 2. 入场时机
- **价格突破**：突破关键价位
- **支撑阻力**：在支撑位买入，阻力位卖出
- **订单流**：观察大单和订单簿

### 3. 持仓时间
- **秒级**：几秒到几十秒
- **分钟级**：几分钟内完成
- **快速止盈**：达到目标立即离场

### 4. 出场策略
- **固定点数**：达到预设点数立即离场
- **快速止损**：亏损立即止损
- **时间止损**：超过预定时间未盈利就离场

## 实战应用

### 适用市场
- ✅ 高流动性市场（外汇、期货）
- ✅ 交易成本低的市场
- ✅ 波动适中的市场
- ❌ 流动性差的市场
- ❌ 交易成本高的市场

### 优势
1. **风险暴露短**：持仓时间短，风险小
2. **不受隔夜风险**：日内平仓
3. **机会多**：每天有大量交易机会
4. **可控性强**：快速止损止盈

### 劣势
1. **交易成本高**：频繁交易导致手续费累积
2. **心理压力大**：需要高度集中注意力
3. **时间投入大**：需要盯盘
4. **技术要求高**：需要快速执行

## 心理要求

1. **专注力**：高度集中注意力
2. **快速反应**：瞬间决策能力
3. **情绪控制**：不因小亏损影响心态
4. **耐心**：等待最佳时机

## 风险管理

1. **严格止损**：每笔亏损极小
2. **控制频率**：避免过度交易
3. **成本控制**：选择低手续费平台
4. **精力管理**：避免疲劳交易
