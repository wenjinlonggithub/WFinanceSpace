# 仓位管理哲学

## 核心思想

**"资金管理决定生死"**

仓位管理不是一种交易方向的哲学，而是关于如何管理资金和风险的哲学。无论采用什么交易策略，优秀的仓位管理都是长期生存和盈利的关键。

### 哲学基础

1. **生存第一**：保护资本是首要任务
2. **风险控制**：控制每笔交易的风险
3. **资金曲线**：追求平滑的资金曲线
4. **复利增长**：通过合理仓位实现复利

## 战法原理

### 1. 固定比例法
- **固定百分比**：每笔交易风险固定（如1%）
- **简单有效**：最常用的方法
- **适合新手**：容易理解和执行

### 2. 凯利公式
- **数学优化**：根据胜率和赔率计算最优仓位
- **理论最优**：长期收益最大化
- **实际应用**：通常使用半凯利或1/4凯利

### 3. 金字塔加仓
- **盈利加仓**：只在盈利时加仓
- **风险递减**：后续仓位风险更小
- **趋势跟踪**：适合趋势交易

### 4. 马丁格尔（不推荐）
- **亏损加仓**：亏损时加倍仓位
- **高风险**：可能导致爆仓
- **赌徒心态**：违背风险管理原则

## 历史案例

### 拉里·威廉姆斯
- **哲学**：激进的仓位管理
- **案例**：1年将1万美元变成110万美元
- **教训**：高收益伴随高风险

### 长期资本管理公司（LTCM）
- **失败案例**：过度杠杆导致破产
- **教训**：即使策略正确，过度杠杆也会致命

## 实战应用

### 风险等级

**保守型（1%风险）**
- 单笔交易风险：账户的1%
- 最大回撤：10-15%
- 适合：新手、稳健投资者

**中等型（2%风险）**
- 单笔交易风险：账户的2%
- 最大回撤：20-25%
- 适合：有经验的交易者

**激进型（3-5%风险）**
- 单笔交易风险：账户的3-5%
- 最大回撤：30-40%
- 适合：专业交易者

### 仓位计算

```
仓位大小 = (账户资金 × 风险百分比) / 每股风险金额
```

**示例**：
- 账户：10万元
- 风险比例：1%
- 风险金额：1000元
- 入场价：50元
- 止损价：48元
- 每股风险：2元
- 仓位：1000 / 2 = 500股

## 心理要求

1. **纪律**：严格执行仓位规则
2. **理性**：不因情绪改变仓位
3. **耐心**：接受小仓位的慢增长
4. **谦逊**：承认市场的不确定性

## 风险管理原则

### 1. 单笔风险控制
- 每笔交易风险不超过账户的1-2%
- 连续亏损不会致命

### 2. 总仓位控制
- 总持仓不超过账户的50-70%
- 保留现金应对机会和风险

### 3. 相关性控制
- 避免持有高度相关的品种
- 真正的分散投资

### 4. 动态调整
- 根据市场环境调整仓位
- 震荡市减仓，趋势市加仓

## 常见错误

1. **满仓交易**：一次失败就致命
2. **亏损加仓**：越亏越多
3. **重仓单一品种**：缺乏分散
4. **忽视相关性**：假分散
